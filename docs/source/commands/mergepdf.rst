Merge PDFs
==========

Category
--------
PDF Commands

Description
-----------

This command merge multiple pdf into single file. 

Parameters
----------

**PDF File List**
	Enter the file list and separate by comma ex. C:\temp\myfile1.pdf,C:\temp\myfile2.pdf

**Result PDF File**
	Indicate the result file (ex. C:\\temp\\myzip\\mypdf.pdf
