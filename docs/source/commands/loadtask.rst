Load Task
=========

Category
--------
Task Commands

Description
-----------

This command pre-loads tasks for future execution. Use this command when you want to load a task but not immediately execute it.

Parameters
----------

**Task Path**
	Select a task to load. After, use 'Run Task' with the same path to execute.
