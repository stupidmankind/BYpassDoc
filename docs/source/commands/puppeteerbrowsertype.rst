PPT Browser Type
================

Category
--------
Web Browser Commands

Description
-----------

This command allows you to type into a web browser session.

Parameters
----------

**Instance Name**
	Enter the instance name

**Frame**
	Specify Frame Name.

**XPATH**
	Specify which XPATH to look at.

**Text**
	Specify Text to type.

**Encryption**
	Indicate if the text in 'TextToSend' is Encrypted.

**Clear Text**
	Specify Text to type.
