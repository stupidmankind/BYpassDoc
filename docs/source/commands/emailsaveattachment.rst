Save Attachment
===============

Category
--------
Email Commands

Description
-----------

This command save the emails have attactment.

Parameters
----------

**Message Variable**
	Please Enter the IMAP email variable

**Attachments Save Folder**
	Please specify the folder name



Returned Value
--------------
	This command does not have returned value.
