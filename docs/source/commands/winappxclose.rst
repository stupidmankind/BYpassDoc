WinAppX Close
=============

Category
--------
WinApp Automation Commands

Description
-----------

Close WinAppX.

Parameters
----------

**Instance Name**
	Enter the instance name.

**Close App**
	Also close Application when finish.



Returned Value
--------------
	This command does not have returned value.

Example Usage
-------------

	Example Location:  
		`BYpass\\Examples\\WinApp Automation Commands\\WinAppX Close\\WinAppX Close.xml`
