List Remove Duplicate
=====================

Category
--------
List Commands

Description
-----------

This command allows you to remove duplicate in List. Use this command when you want to remove duplicate in a List.

Parameters
----------

**List Name**
	Enter List Name



Returned Value
--------------
	This command does not have returned value.
