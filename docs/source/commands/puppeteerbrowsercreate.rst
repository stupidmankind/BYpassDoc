PPT Browser Create
==================

Category
--------
Web Browser Commands

Description
-----------

This command allows you to create a new web browser session.

Parameters
----------

**Instance Name**
	Enter the instance name

**Browser**
	Select the browser

**Headless**
	Open browser in background

**Maximized**
	Open browser in maximized window

**URL**
	Please Enter the URL to navigate to
