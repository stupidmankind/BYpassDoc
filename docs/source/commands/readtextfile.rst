Read Text File
==============

Category
--------
Text File Commands

Description
-----------

This command reads text data into a variable. Use this command when you want to read data from text files.

Parameters
----------

**File**
	Indicate the path to the file

**Read Option**
	Select the read type

**Result**
	Indicate the variable to receive the result
